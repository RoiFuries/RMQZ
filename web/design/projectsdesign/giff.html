Gif Search

The goal was to learn React and APIs. This GIF searcher demonstrates my ability to build functional and responsive web solutions.

As a personal project, the challenge was technical: to master the integration with an external API and build a functional web application from scratch. The objective was to create a GIF searcher that was fast and responsive, using a modern stack built around React. The main challenge wasn't just to make it work, but to make the learning process clear and methodical.
Our Approach
I approached this project as a learning and development exercise. My focus was to build a prototype that would validate the idea of consuming an API, making the decision to use React.js for the frontend development. The process allowed me to understand state management, handling requests, and data manipulation, applying a structured mindset to each step to ensure a fluid execution.
The Solution
The result is a fully functional GIF searcher that demonstrates my skills in React development and in consuming APIs efficiently. This project not only fulfilled its learning objective but also became a tangible proof of my ability to build web solutions from their foundations and to tackle new technological challenges.
